# Kerala Jersey Backend

A FastAPI project initialized with `uv`.

## Setup

This project uses `uv` for dependency management.

1. Install dependencies:
   ```bash
   uv sync
   ```

2. Run the development server:
   ```bash
   uv run uvicorn main:app --reload
   ```
